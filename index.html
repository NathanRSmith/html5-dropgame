Instructions

To play this game click on blocks in groups of 2 or more of the same color. Larger groups are worth many more points than smaller ones (n2-n).

Once there are no more moves the game is over. A total score is computed by taking the game score minus a score penalty (10 x number left).

Click on the button to start a new game at any time. The button will display statistics of completed games played.
